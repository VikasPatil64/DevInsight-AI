# DevInsight AI 🚀  
### AI-powered GitHub Developer Analyzer

An AI-powered developer profile analyzer that evaluates GitHub profiles and generates insights on performance, activity, and skill trends.

## 🔥 Features

- 📊 GitHub Profile Analysis
- ⭐ Developer Scoring System
- 📈 Repository Insights (stars, forks, languages)
- 📉 Activity Tracking (last active, consistency)
- 🤖 AI-generated professional summary (with fallback handling)

## 🛠️ Tech Stack

- Backend: FastAPI
- Frontend: HTML, CSS, JS
- APIs: GitHub API, OpenRouter (LLM)

## ⚙️ How it works

1. Enter GitHub username
2. Fetch profile & repositories
3. Analyze metrics
4. Generate developer score
5. Display insights dashboard

## ⚠️ Note

AI summary may be temporarily unavailable due to free API limitations. Core analytics remain fully functional.
